===== PAGE 7 =====
Stage 2: Pre-training
Self-Supervised Learning Objectives:
Causal Language Modeling (CLM):
Predict next token given previous tokens
Used in: GPT, LLaMA, PaLM
Objective: max P
t logP(x t |x<t)
Masked Language Modeling (MLM):
Predict masked tokens from context
Used in: BERT, RoBERTa
Bidirectional context
Span Corruption:
Predict corrupted text spans
Used in: T5, UL2
Flexible masking strategies

===== PAGE 11 =====
Alignment: RLHF
Reinforcement Learning from Human Feedback
1 Supervised Fine-tuning (SFT):
Train on high-quality demonstrations
2 Reward Model Training:
Collect human preference comparisons
Train model to predict preferences
Reward function: helpful, harmless, honest
3 RL Optimization:
Use PPO (Proximal Policy Optimization)
Maximize reward while staying close to SFT model
KL divergence penalty
Used in: ChatGPT, Claude, GPT-4

===== PAGE 14 =====
Model Optimization for Deployment
Efficiency Techniques:
Quantization:
INT8, INT4, binary weights
Reduces memory and compute
Trade-off: slight accuracy loss
Pruning:
Remove less important weights
Structured or unstructured
Maintain performance with fewer parameters
Distillation:
Train smaller student model
Mimic larger teacher model
Example: DistilBERT (66% size, 97% performance)
Caching & Batching:
KV cache for attention
Dynamic batching for throughput
